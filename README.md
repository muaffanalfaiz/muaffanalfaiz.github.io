## Featured Projects

### South Sumatra Price Spike Early Warning
Satellite-informed early-warning system for horticultural price spikes using market data, rainfall, vegetation indices, and machine learning.

Repository:  
https://github.com/muaffanalfaiz/south-sumatra-price-spike-early-warning

### Degraded Peatland Agrivoltaic Suitability
GIS-MCDA-based land suitability analysis for agrivoltaic development on degraded peatlands in South Sumatra.

Repository:  
https://github.com/muaffanalfaiz/degraded-peatland-agrivoltaic-suitability

### Tasman Soil Index
Multi-criteria spatial analysis for soil infrastructure suitability mapping in Tasman District, New Zealand.

Repository:  
https://github.com/muaffanalfaiz/tasman-soil-index

### Farm Fuel & DEF Demand NZ
End-to-end data pipeline and forecasting workflow for New Zealand tractor fleet analysis, diesel demand estimation, DEF demand modeling, and scenario-based forecasting.

Repository:  
https://github.com/muaffanalfaiz/farm-fuel-def-demand-nz
